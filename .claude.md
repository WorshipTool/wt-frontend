# Claude Context – Frontend Only

This repository is FRONTEND ONLY.

## Stack

- Next.js
- TypeScript
- Jest for tests

---

## Core Rules

- This is UI / presentation code only
- No backend or database logic here
- All data comes from backend API

---

## API Usage

- API types are GENERATED from backend Swagger
- Generated files are READ-ONLY
- Never modify generated API types manually

---

## Testing Rules (IMPORTANT)

- NEVER run or suggest running E2E tests
- E2E tests are slow and not part of normal development
- Use ONLY:
  - Jest unit tests
  - Jest component tests (if applicable)

Assume:

- E2E tests are handled separately (CI or manual runs)
- Fast feedback is preferred over full coverage

---

## Working Style

- Focus on correctness and clarity
- Prefer simple solutions over complex abstractions
- Avoid unnecessary refactors
